Online Paluwagan App

Android project files with updated Gradle configuration

Download Project (ZIP)

Setup Instructions:

  1. Extract the downloaded ZIP file
  2. Open Android Studio
  3. Click "Open an Existing Project"
  4. Navigate to and select the extracted folder
  5. Wait for Gradle sync to complete
  6. Create a Firebase project and download google-services.json
  7. Replace app/google-services.json with your file
  8. Run the app on your device or emulator
